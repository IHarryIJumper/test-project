# Test Project

## Local project launch

### Installing

Install [Webpack](http://webpack.github.io):

```
npm install webpack -g
```

Install project packages:

```
npm install
```

### Building

Development build:

```
npm run dev
```

Production build:

```
npm run build
```


### Start project

```
npm start
```

### Tests

```
npm tests
```

or

```
npm tests:watch
```

### Project access

```
http://localhost:3000/
```
